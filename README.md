# 🛠️ Guia de Suporte Técnico N1

Este projeto simula atividades reais de um analista de suporte técnico N1, com foco em resolução de problemas comuns do dia a dia.

---

## 📌 Problemas abordados:

### 🌐 Sem acesso à internet
**Diagnóstico:**
- Verificar conexão do cabo ou Wi-Fi
- Testar comando: ipconfig
- Verificar IP e DNS

**Solução:**
- Reiniciar roteador
- Renovar IP (ipconfig /release e /renew)

---

### 💻 Computador lento
**Diagnóstico:**
- Verificar programas em segundo plano
- Checar uso de CPU e memória

**Solução:**
- Finalizar tarefas desnecessárias
- Limpar arquivos temporários

---

### 🔐 Problemas de login
**Diagnóstico:**
- Verificar usuário e senha
- Testar conexão com servidor

**Solução:**
- Redefinir senha
- Verificar rede

---

## 💻 Ferramentas utilizadas:
- Prompt de comando (CMD)
- Conceitos básicos de rede
- Windows

---

## 🎯 Objetivo:
Demonstrar conhecimentos práticos em suporte técnico e troubleshooting.
